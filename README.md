## 🛠️ Getting Started

### 📋 Prerequisites
You need a C compiler installed on your local machine:
* **Linux/macOS**: `gcc` or `clang` 
* **Windows**: MinGW-w64, MSYS2, or WSL (Windows Subsystem for Linux)

### 💻 Compile & Run in a Flash
Navigate to the desired directory and execute this terminal shortcut:

```bash
# Compile and run immediately 🚀
gcc program.c -o program && ./program

```
